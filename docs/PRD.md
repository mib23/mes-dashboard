# 服装生产工厂 MES 数据大屏 PRD

## 1. 产品概述

### 1.1 背景

随着服装制造业的数字化转型，传统的生产管理模式已无法满足实时监控与精细化管理的需求。为了提升工厂的生产效率、降低不合格率并实现管理透明化，本项目旨在开发一套适用于服装生产工厂的数据可视化大屏系统。

### 1.2 目标

- **全局掌控**：为工厂管理层提供全厂生产、质量、设备及能耗的宏观视图。
- **精益生产**：为各生产车间（编织、套口、缝合、后整包装等）提供实时的产量目标进度、瓶颈工序及异常报警，指导现场快速响应。
- **数据驱动**：打破信息孤岛，集成现有 ERP/MES 数据，实现车间看板无纸化、自动化。

## 2. 用户角色

- **厂长/生产总监**：关注工厂总览大屏，侧重订单交付率、全厂综合产能、成本与良率。
- **车间主任/组长**：关注各车间专属大屏，侧重当班产量、工序平衡、人员效率及异常处理。
- **一线操作工/质检员**：被动接受大屏信息（如目标提醒、不良率警示）。

## 3. 功能需求详细说明

### 3.1 工厂总大屏（管理层视图）

**核心诉求**：宏观指标监控，辅助高层决策。

- **全厂生产概览**：今日计划产量 vs 实际完成量（进度条/仪表盘）；整体达成率及环比/同比分析。
- **急单/重点订单追踪**：滚动显示 Top 5 紧急交货订单的当前工序及预计完工时间。
- **全厂质量看板**：整体一次交验合格率；各车间次品率排名（柱状图）。
- **设备运行状态**：全厂核心设备（如电脑横机、自动套口机、洗水设备）的开机率、故障停机数量。
- **车间效能红黑榜**：各车间（编织、套口、缝合、后整包装等）的 OEE（设备综合效率）或人均产能对比。

### 3.2 编织车间大屏（核心设备车间）

**核心诉求**：横机运转率与纱线利用率。

- **编织生产进度**：计划织片数量 vs 实际完成织片数量（前片、后片、袖子等）。
- **设备运行状态看板**：全车间电脑横机（按机区划分）的实时状态图（绿色-运行，黄色-待机，红色-断纱/故障停机）。
- **异常预警**：断纱报警、漏针预警、针板保养提醒。
- **设备综合效率（OEE）**：实时计算核心机台的稼动率、性能开动率及合格率。

### 3.3 套口与缝合车间大屏（核心人工作业车间）

**核心诉求**：工序平衡、瓶颈突破与实时质量控制。

- **班组产能看板**：各班组（套口组、挑撞组、平缝组）当班目标产量、当前实际产量、达成率差异。
- **工序平衡与瓶颈分析**：实时显示各工序（如套领、套袖、侧缝等）的滞留件数（WIP在制品）；标红预警堆积最严重的“瓶颈工序”。
- **实时质量警示**：巡检发现的 Top 3 质量缺陷（如漏套、脱针、跳线）及频次；预警：若某工位连续出现 3 件同类不良，大屏闪烁报警。
- **员工效能排行榜**：实时滚动显示计件工资/产量 Top 5 员工（如套口熟练工）及倒数预警。

### 3.4 后整车间大屏（洗水、熨烫、包装）

**核心诉求**：后处理工艺控制与成品入库进度。

- **洗水/烘干进度与监控**：洗水机/烘干机运行状态，水温、洗涤时间等关键工艺参数监控（如缩水率控制相关数据）。
- **整烫与包装进度**：待整烫/待包装件数 vs 已完成件数。
- **订单齐套率**：按订单/颜色/尺码显示齐套入库情况，防止短装或混码。
- **物流接驳状态**：待出货成品堆区状态及发货进度。

## 4. 非功能性需求

### 4.1 交互与视觉设计（UI/UX）

- **高对比度**：车间环境光线复杂，需采用深色背景（科技蓝/暗黑），辅以高饱和度的红/绿/黄作为状态色。
- **大字号**：关键数据指标（如产量、达成率）需保证在 10-15 米外清晰可见。
- **自动轮播**：对于信息量较大的图表（如几十条产线的数据），支持定时自动翻页/平滑滚动。

### 4.2 性能与实时性

- **数据延迟**：核心数据（产量、报警）延迟不超过 5 秒（建议采用 WebSocket 推送或高频轮询）。
- **高可用性**：大屏前端程序需具备断线自动重连、内存泄漏防护机制，保证 7x24 小时长期亮屏不卡顿、不白屏。

### 4.3 硬件适配

- 适配主流智能电视或商用显示器分辨率（标准 1920x1080，支持 4K 3840x2160 响应式缩放）。

## 5. 接口与数据源要求

- 需与工厂现有的 ERP（获取订单与生产计划）、MES（获取工单进度、报工数据、质检数据）、物联网 IoT 平台（获取设备状态、智能吊挂系统数据）进行接口打通（RESTful API / WebSocket / MQTT）。
